# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-04-01

### Added

- **12 skills** covering the full SOC lifecycle: auth, incidents, event queries, CMDB, investigations, threat hunting (quick + hypothesis-driven), IOC management, UEBA, rule creation, rule management, playbooks, multi-org operations, compliance, ticketing
- **28 slash commands** for instant access to all workflows
- **3 specialized agents**: SOC Analyst (L1/L2), Rule Engineer, Threat Hunter (L3)
- **7 Python scripts**: API wrapper, investigation pipeline, AI reasoning engine, IOC hunter, UEBA reporter, PDF report generator, scheduled hunt automation
- **5 IR playbooks**: ransomware, account compromise, data exfiltration, malware, insider threat
- **15 production rule templates** with MITRE ATT&CK mapping
- **Docker container** with plugin pre-installed
- **Multi-org / MSSP support** for Service Provider deployments
- **Autonomous triage engine** combining asset criticality, FP rates, and threat intel
- **Advanced hunting techniques**: beacon analysis, DNS long-tail, stack counting, impossible travel, Office process tree analysis
- **APT group profiles**: APT28, APT29, Lazarus, FIN7, Sandworm, APT41
- **Compliance reporting**: PCI DSS, HIPAA, SOX, NIST, ISO 27001
- **ITSM integration**: ServiceNow, Jira, PagerDuty ticket creation
